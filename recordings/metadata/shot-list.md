# Recording Shot List

## Long Walkthrough, 6-8 Minutes

1. Terminal: run `./scripts/live-pipeline-up.sh` and show batch-1 ingestion
   metrics, vector-count delta, throughput, and estimated cost.
2. Director view: open password-protected portal and choose Director.
3. Search: run the selected query and show ranked images, object keys, source
   metadata, and pipeline latency.
4. Terminal: run `./scripts/live-ingest-batch.sh 2` and show the second metric
   readout.
5. Search again: rerun the same query and call out new image keys/results from
   the appended batch.
6. PI/RSE views: show benchmark, operations, cache, DLQ, provider abstraction,
   Ray/Databricks paths, and deployment view.
7. Scale evidence: show recorded benchmark output, Qdrant/Ray metrics, and cost
   model assumptions.
8. Close: state partner value and where a project collaboration would start.

## Teaser Cut, 2 Minutes

1. One-sentence project context.
2. Batch-1 metric readout.
3. Search result gallery with latency.
4. Batch-2 append and same-query result change.
5. Benchmark/cost chart.
6. CSSE value proposition.
